tips

hexo现在支持更加简单的命令格式了，比如：

  • hexo g == hexo generate

  • hexo d == hexo deploy

  • hexo s == hexo server

  • hexo n == hexo new

  • hexo g -d == hexo g + hexo d // 每次更新配置或者文章都需要使用一次
